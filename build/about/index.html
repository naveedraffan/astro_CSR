About this project

A quick-start Astro CSR template with Yarn and a custom build output directory.

Build configuration

// astro.config.mjs
export default defineConfig({
  output: 'static',   // CSR / SSG mode
  outDir: './build',  // custom output directory
  build: {
    assets: '_assets',
    inlineStylesheets: 'auto',
  },
});

Run yarn build → all static files land in ./build/. Serve that folder from any CDN or static host (Netlify, Vercel, GitHub Pages, S3 + CloudFront…).
